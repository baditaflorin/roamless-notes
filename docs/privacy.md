# Privacy

Roamless Notes has no analytics in v1.

The app stores notes, selected block metadata, settings, indexes, and generated embeddings in the user's browser storage. Semantic model files are fetched by the browser only when the user starts a local AI action. No note content is sent to a project server because there is no project server.
